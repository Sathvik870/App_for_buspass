Steps to Apply for Bus Pass

Required Documents for details

  • Valid College ID Card
  • Proof of Address (Aadhaar/Ration Card)
  • Passport size photograph
  • Student signature

Application Process

  1. Fill in personal details
  2. Upload required documents
  3. Verify travel route
  4. Make payment
  5. Download bus pass

Steps to Renew Bus Pass

Renewal Process

  1. Enter existing Student roll number
  2. Enter existing Student DOB/li>
  3. Verify current details
  4. Make renewal payment
  5. Download renewed bus pass

Important Notes

  • Verify all details are up to date
  • Keep previous pass for reference
